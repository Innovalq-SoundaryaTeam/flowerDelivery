## Overview

Flora Daily is a responsive front-end e-commerce website designed for online flower and gift delivery. The project offers a clean, modern user interface with shopping features such as product browsing, wishlist, shopping cart, checkout, user authentication pages, and dashboard pages.

- Responsive Design
- Modern User Interface
- Dark Mode
- Product Catalog
- Product Details
- Wishlist
- Shopping Cart
- Checkout
- User Login & Registration
- Forgot Password
- Dashboard
- Order Tracking
- Order History
- Blog
- Contact Form Validation
- 404 Error Page
- Coming Soon Page

## Installation

1. Download or extract the project.
2. Open the folder in Visual Studio Code.
3. Open `pages/index.html`.
4. Launch the project using the **Live Server** extension.

No backend setup or package installation is required.
