Project Description
-------------------
Flora Daily is a modern flower and gift delivery website developed using
HTML5, CSS3, Bootstrap 5, and JavaScript. The website provides an elegant
shopping experience with features like product browsing, wishlist,
shopping cart, checkout, authentication pages, dashboard pages, and
responsive design.
