Installation Steps
---------------------------------------------------------

Step 1

Extract the project ZIP file.

Step 2

Open the project folder using Visual Studio Code.

Step 3

Install the Live Server extension if it is not already installed.

Step 4

Open:

pages/index.html

Step 5

Right-click the file and select:

Open with Live Server

OR

Click the "Go Live" button in the bottom-right corner.

The project will open automatically in your default browser.

Troubleshooting
---------------------------------------------------------

If CSS or JavaScript is not loading:

• Ensure the folder structure is unchanged.
• Check relative file paths.
• Run the project using Live Server.

If images are missing:

• Verify the image exists inside:

assets/images/

• Ensure the filename matches exactly.
